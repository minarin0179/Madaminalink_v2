# 導入ガイド

このガイドでは、Madaminalinkを導入して使い始めるための手順を説明します。

## 📥 Botの招待

### 必要な権限

Madaminalinkは以下の権限が必要です：

- チャンネルの閲覧
- チャンネルの管理
- ロールの管理
- メッセージの送信
- メッセージの管理
- メッセージ履歴の閲覧
- リンクの埋め込み
- ファイルの添付
- スレッドの作成と管理
- ボイスチャンネルへの接続とメンバーの移動

### 招待リンク

以下のリンクからBotをサーバーに招待できます：

[Botを招待する](https://discord.com/api/oauth2/authorize?client_id=926051893728403486&permissions=8&scope=bot%20applications.commands)

⚠️ **注意**: 上記リンクは管理者権限を付与します。必要に応じて権限を調整してください。

## 🎮 初めての使用

### 1. スラッシュコマンドの確認

Botを招待すると、自動的にスラッシュコマンドが登録されます。チャットで `/` を入力すると利用可能なコマンド一覧が表示されます。

### 2. 基本的なセットアップ

マーダーミステリーゲームを始める場合、以下の手順を推奨します：

```
/setup
```

このコマンドで、ゲーム用のカテゴリとチャンネルを自動作成できます。

### 3. 主要コマンドの使い方

#### チャンネル管理
- `/setup` - ゲーム用カテゴリとチャンネルの自動作成
- `/copy` - チャンネルやカテゴリをコピー
- `/delete` - カテゴリ内のチャンネルを一括削除
- `/cleanup` - チャンネル内のメッセージを全削除

#### ゲーム機能
- `/poll` - キャラクター選択や投票を実施
- `/dice` - ダイスロールボタンを作成
- `/order` - 順序管理（発言順など）
- `/remind` - リマインダーの設定

#### アーカイブ
- `/archive` - カテゴリやチャンネルをスレッドにアーカイブ

## 🛡️ 安全機能

### サーバー参加後の制限

Botがサーバーに参加してから24時間は、以下の破壊的なコマンドが実行できません：

- `/delete`
- `/cleanup`
- `/archive`

これは誤操作を防ぐための安全措置です。

### 権限の確認

各コマンドを実行する前に、Botが必要な権限を持っているか自動的に確認されます。権限が不足している場合はエラーメッセージが表示されます。
