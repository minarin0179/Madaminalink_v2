# /sync

カテゴリ内のすべてのチャンネルの権限をカテゴリと同期します。カテゴリに権限を設定してから同期することで、一括で権限設定ができます。

## 使用方法

```
/sync 同期するカテゴリ:#カテゴリ名
```

## オプション

| オプション | 必須 | 説明 |
|-----------|------|------|
| 同期するカテゴリ | はい | 権限を同期するカテゴリ |

## 使用例

### 権限を同期

```
/sync 同期するカテゴリ:#セッション用
```

カテゴリに設定された権限が、中のすべてのチャンネルに適用されます。

## 用途

### ケース1: カテゴリ全体の権限変更

1. カテゴリの権限設定を変更
2. `/sync` を実行
3. カテゴリ内の全チャンネルに権限が反映される

### ケース2: チャンネル追加後の権限統一

1. カテゴリにチャンネルを追加
2. 追加したチャンネルの権限がカテゴリと異なる場合
3. `/sync` で権限を統一

## 注意事項

::: tip ヒント
個別に設定したチャンネルの権限は、同期によりカテゴリの権限で上書きされます。
:::
