# /setup

新規プレイ用のカテゴリ、チャンネル、ロールを一括作成します。

## 使用方法

```
/setup シナリオ名:シナリオ名 プレイヤー数:4 密談チャンネル数:2
```

## オプション

| オプション | 必須 | 説明 |
|-----------|------|------|
| シナリオ名 | はい | カテゴリ・ロールの名前に使用 |
| プレイヤー数 | はい | 個別チャンネル・ロールの数 |
| 密談チャンネル数 | はい | 密談用VCの数 |
| ロールの作成位置 | いいえ | 指定したロールの下に新規ロールを作成 |
| 個別ロールを作成しない | いいえ | 個別チャンネルは作るがロールは作成しない |
| キャラ名を指定 | いいえ | キャラクター名をスペース区切りで指定 |

## 作成されるもの

### カテゴリ構成
- シナリオ名（カテゴリ）
  - 一般（テキスト）
  - 共通情報（テキスト）
  - 観戦（テキスト）
  - PC1〜PCn（テキスト × プレイヤー数）
  - 解説（テキスト）
  - 全体会議（VC）
  - 密談場所1〜n（VC × 密談チャンネル数）

### ロール構成
- シナリオ名 GM
- シナリオ名 PL
- シナリオ名 観戦
- シナリオ名 PC1〜PCn（個別ロール）

## 使用例

### 基本的な使い方

```
/setup シナリオ名:マーダーミステリー プレイヤー数:6 密談チャンネル数:3
```

### キャラ名を指定

```
/setup シナリオ名:マーダーミステリー プレイヤー数:4 密談チャンネル数:2 キャラ名を指定:探偵 助手 医者 執事
```

PC1〜PCnの代わりに指定したキャラ名でチャンネル・ロールが作成されます。

### 個別ロールなしで作成

```
/setup シナリオ名:マーダーミステリー プレイヤー数:4 密談チャンネル数:2 個別ロールを作成しない:はい
```

個別チャンネルは作成されますが、個別ロールは作成されません。

## 注意事項

::: warning 注意
チャンネル数の上限は500個です。プレイヤー数＋密談チャンネル数＋5（固定チャンネル）が50を超える場合はエラーになります。
:::

::: tip ヒント
キャラ名を指定する場合、スペース区切りで入力してください。キャラ名の数はプレイヤー数と一致している必要があります。
:::
