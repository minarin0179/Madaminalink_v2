title: これ以前を削除
description: 選択したメッセージ以前のすべてのメッセージを削除します。
---
<PageHeader />

## 使用方法

1. 削除したい範囲の終点となるメッセージを右クリック
2. 「アプリ」→「これ以前を削除」を選択
   ![コンテキストメニューの「アプリ」サブメニューに表示されるBotコマンド一覧](../images/commands/context-menu/app-submenu.png)
3. 選択したメッセージ自体と、それより前のすべてのメッセージが削除されます

::: danger 削除したメッセージは復元できません
実行前に必要なログがないか確認してください。ログを残したい場合は [/archive](/commands/archive) で保存できます。
:::

::: warning 荒らし対策のため、Botがサーバーに追加されてから24時間は使用できません
:::

## 削除範囲

- **削除対象**: 選択したメッセージ自体と、それより**前**に送信されたすべてのメッセージ
- **選択したメッセージ自体**: 削除されます（残りません）

## 用途

### 特定のメッセージ以前の一括削除

最初の方のメッセージを削除し、途中から残したい場合に便利です。

::: tip チャンネル全体を削除したい場合
チャンネル内のすべてのメッセージを削除したい場合は [/cleanup](/commands/cleanup) コマンドが便利です。
:::
