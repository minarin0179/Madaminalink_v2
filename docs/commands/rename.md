## 使用方法

```
/rename ロール:@ロール名
```

## オプション

| オプション | 必須 | 説明 |
|-----------|------|------|
| ロール | はい | ニックネームを変更するロール |
| 先頭につける文字 | いいえ | ニックネームの先頭に追加する文字（指定しない場合はリセット） |
| 元の名前に戻す | いいえ | 元のユーザー名に戻すか |

## 使用例

### ニックネームをリセット

```
/rename ロール:@観戦
```

「先頭につける文字」を指定しないと、@より前の文字が削除されます。

例: `(観戦)@みなりん` → `みなりん`

### 観戦者マークをつける

```
/rename ロール:@観戦 先頭につける文字:(観戦)
```

指定した文字がニックネームの先頭に追加されます。

例: `みなりん` → `(観戦)@みなりん`

### 通過者マークに変更

```
/rename ロール:@プレイヤー 先頭につける文字:(通過者)
```

既存の先頭文字を置き換えます。

例: `探偵@みなりん` → `(通過者)@みなりん`

## 用途

### ケース1: セッション終了後のニックネームリセット

1. `/rename ロール:@プレイヤー` を実行
2. キャラ名などの先頭文字が削除される
3. 元のニックネームに戻る

### ケース2: 観戦者の識別

1. 観戦者に観戦ロールを付与
2. `/rename ロール:@観戦 先頭につける文字:(観戦)` を実行
3. 観戦者のニックネームに目印がつく

## 注意事項

::: warning 注意
管理者権限を持つメンバーのニックネームは変更できません（Discordの仕様）。変更に失敗したメンバーは一覧で表示されます。
:::

::: tip ヒント
- `@` は全角でも半角でも動作します
- ニックネーム変更時は `@` より前の文字が置き換えられます
- リセット時は `@` より前の文字が削除されます
:::
