## 使用方法

```
/rename ロール:@ロール名
```

実行すると、指定したロールを持つ全メンバーのニックネームが一括で変更されます。

## ニックネーム変更の仕組み

`/rename`コマンドは、ニックネームの`@`を区切りとして動作します。

| 操作 | 変更前 | 変更後 |
|------|--------|--------|
| リセット（先頭文字なし） | `探偵@みなりん` | `みなりん` |
| 先頭文字を追加 | `みなりん` | `(観戦)@みなりん` |
| 先頭文字を置換 | `探偵@みなりん` | `(通過者)@みなりん` |
| 元の名前に戻す | `探偵@みなりん` | ※元のユーザー名 |

::: tip @の扱い
`@`は全角（＠）でも半角（@）でも同様に動作します。複数の`@`がある場合は、最後の`@`より後ろの部分が名前として使用されます。
:::

::: tip 一人だけリセットしたい場合
特定のユーザーのニックネームだけをリセットしたい場合は、ユーザーを右クリック→「アプリ」→「[ニックネームをリセット](/commands/reset-nickname)」が便利です。
:::

## オプション

| オプション | 必須 | 説明 |
|-----------|------|------|
| ロール | はい | ニックネームを変更するロール |
| 先頭につける文字 | いいえ | ニックネームの先頭に追加する文字 |
| 元の名前に戻す | いいえ | 元のユーザー名に戻すか |

### オプションの詳細

#### ロール（必須）

ニックネームを変更する対象のロールを指定します。このロールを持つすべてのメンバーのニックネームが変更されます。

#### 先頭につける文字

ニックネームの先頭に追加する文字を指定します。

- **デフォルト**: 未指定（リセット動作）
- **指定した場合**: `先頭文字@元の名前` の形式に変更
- **未指定の場合**: `@`より前の文字を削除してリセット

::: warning ニックネームの文字数上限
Discordのニックネームは32文字までです。変更後のニックネームが32文字を超える場合、末尾が「…」で省略されます。
:::

#### 元の名前に戻す

「はい」を選択すると、ニックネームを完全にリセットしてDiscordのユーザー名に戻します。

- **デフォルト**: いいえ
- **「はい」の場合**: ニックネームが削除され、Discordのユーザー名が表示される状態になります

## 使用例

### ニックネームをリセット

```
/rename ロール:@プレイヤー
```

「先頭につける文字」を指定しないと、`@`より前の文字が削除されます。セッション終了後にキャラ名を削除する場合などに使用します。

例: `探偵@みなりん` → `みなりん`

### 観戦者マークをつける

```
/rename ロール:@観戦 先頭につける文字:(観戦)
```

指定した文字がニックネームの先頭に追加されます。観戦者をプレイヤーと区別する場合などに使用します。

例: `みなりん` → `(観戦)@みなりん`

### 通過者マークに変更

```
/rename ロール:@プレイヤー 先頭につける文字:(通過者)
```

既存の先頭文字を新しい文字に置き換えます。セッション進行中にステータスを変更する場合などに使用します。

例: `探偵@みなりん` → `(通過者)@みなりん`

::: warning 管理者権限を持つメンバー
管理者権限を持つメンバーのニックネームは変更できません（Discordの仕様）。変更に失敗したメンバーは一覧で表示されます。
:::
