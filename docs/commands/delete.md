## 使用方法

```
/delete 削除するカテゴリ:#カテゴリ名
```

実行すると、指定したカテゴリとその中のチャンネルがすべて削除されます。

::: danger 復元不可
削除したチャンネルとメッセージは復元できません。必要なログは事前に [/archive](/commands/archive) で保存してください。
:::

::: tip メッセージのみを消去したい場合
カテゴリやチャンネルは残してメッセージのみを消去したい場合は [/cleanup](/commands/cleanup) を使用してください。
:::

## オプション

| オプション | 必須 | 説明 |
|-----------|------|------|
| 削除するカテゴリ | はい | 削除するカテゴリ |
| ロールの削除 | いいえ | カテゴリに付与されたロールも削除するか |

### オプションの詳細

#### 削除するカテゴリ（必須）

削除するカテゴリを指定します。カテゴリ内のすべてのチャンネルも一緒に削除されます。

#### ロールの削除

カテゴリ内のチャンネルに権限設定されているロールを一緒に削除します。

- **デフォルト**: いいえ（ロールは削除しない）
- **指定可能な値**: はい / いいえ
- **削除対象**: カテゴリ内の各チャンネルに権限が設定されているロール（@everyoneは除く）

::: warning ロール削除の注意
カテゴリ内のチャンネルに権限が設定されているロールはすべて削除対象になります。複数のセッションでロールを使いまわしている場合は、このオプションを指定しないでください。マダミナリンクより上位のロールは削除できません。削除に失敗したロールがある場合はその旨が表示されます。
:::

## 使用例

### カテゴリのみ削除

```
/delete 削除するカテゴリ:#終了したセッション
```

カテゴリとその中のチャンネルをすべて削除します。ロールは残ります。

### ロールも一緒に削除

```
/delete 削除するカテゴリ:#終了したセッション ロールの削除:はい
```

カテゴリ、チャンネルに加えて、関連するロールも一緒に削除します。

## 用途

### セッション終了後の片付け

1. セッション終了後、必要に応じて [/archive](/commands/archive) でログを保存
2. `/delete 削除するカテゴリ:#カテゴリ名 ロールの削除:はい` でカテゴリとロールを一括削除
