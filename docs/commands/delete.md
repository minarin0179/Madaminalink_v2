# /delete

カテゴリとその中のチャンネルをすべて削除します。セッションごとにロールを分けて運用している場合、ロールも一緒に削除できます。

## 使用方法

```
/delete 削除するカテゴリ:#カテゴリ名
```

## オプション

| オプション | 必須 | 説明 |
|-----------|------|------|
| 削除するカテゴリ | はい | 削除するカテゴリ |
| ロールの削除 | いいえ | カテゴリに付与されたロールも削除するか（デフォルト: いいえ） |

## 使用例

### カテゴリのみ削除

```
/delete 削除するカテゴリ:#終了したセッション
```

カテゴリとその中のチャンネルをすべて削除します。

### ロールも一緒に削除

```
/delete 削除するカテゴリ:#終了したセッション ロールの削除:はい
```

カテゴリに関連付けられたロールも一緒に削除します。

## ユースケース

### ケース1: セッション終了後のクリーンアップ

1. セッション終了後、必要に応じて `/archive` でログを保存
2. `/delete` でカテゴリを削除
3. 「ロールの削除」を「はい」にして、セッション用ロールも削除

## 注意事項

::: danger 危険
削除したチャンネルとメッセージは復元できません。必要なログは事前に `/archive` で保存してください。
:::

::: warning 注意
マダミナリンクより上位のロールは削除できません。削除に失敗した場合はその旨が表示されます。
:::

::: tip ヒント
テキストチャンネル内のメッセージのみを消去したい場合は `/cleanup` を使用してください。
:::

## 関連コマンド

- [/archive](/commands/archive) - 削除前にログを保存
- [/cleanup](/commands/cleanup) - メッセージのみを削除
- [/setup](/commands/setup) - 新規カテゴリを作成
