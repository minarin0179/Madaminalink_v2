# /remind

指定した日時にメッセージを自動送信するリマインダーを設定します。セッションの告知やリマインドに活用できます。

## 使用方法

```
/remind set
```

## サブコマンド

| サブコマンド | 説明 |
|-------------|------|
| set | 新しいリマインダーを登録 |
| list | 登録済みのリマインダーを確認 |

## オプション（set）

| オプション | 必須 | 説明 |
|-----------|------|------|
| 月 | はい | 送信する月（1〜12） |
| 日 | はい | 送信する日（1〜31） |
| 時 | はい | 送信する時（0〜23） |
| 分 | はい | 送信する分（0〜59） |
| 本文 | はい | 送信するメッセージ内容 |
| 送信先 | いいえ | 送信先チャンネル（デフォルト: 入力したチャンネル） |

## 使用例

### リマインダーを設定

```
/remind set 月:12 日:25 時:20 分:0 本文:本日20時からセッション開始です！
```

指定した日時にメッセージが自動送信されます。

### 送信先を指定

```
/remind set 月:12 日:25 時:20 分:0 本文:セッション開始！ 送信先:#告知
```

指定したチャンネルにリマインダーを送信します。

### リマインダー一覧を確認

```
/remind list
```

登録済みのリマインダーが表示され、不要なものは削除できます。

## 用途

### ケース1: セッション前日のリマインド

1. `/remind set` でセッション前日にリマインダーを設定
2. メンションを含めて参加者に通知
3. 指定日時に自動でメッセージが送信される

## 注意事項

::: warning 注意
- 過去の日付や3ヶ月以上先の日付は設定できません
- 過去の日付を指定した場合、自動的に翌年に設定されます
:::

::: tip ヒント
メッセージ内に任意のメンションを含めることができます。
:::
