# /role

ロールの付与・解除を行うボタンを作成します。

## 使用方法

```
/role [サブコマンド]
```

## サブコマンド

| サブコマンド | 説明 |
|-------------|------|
| add | 対象のロールを持つメンバーに別のロールを付与 |
| remove | 選択したロールをすべてのメンバーから解除 |
| change | ロールを別のロールに付け替え |
| self | ロールの取得・解除を行うボタンを設置 |

## 使用例

### ロールを付け替える（change）

```
/role change 解除するロール:@PL 付与するロール:@通過者
```

ボタンを押すと、PLロールが解除され、通過者ロールが付与されます。

### ロールを付与する（add）

```
/role add 対象:@PL 付与するロール:@参加済み
```

PLロールを持つ全メンバーに参加済みロールを付与するボタンを作成します。

### ロールを解除する（remove）

```
/role remove 解除するロール:@PL
```

PLロールを全メンバーから解除するボタンを作成します。

### セルフロールボタン（self）

```
/role self 付与するロール:@参加希望
```

押すとロールを取得・解除できるボタンを設置します。

### ロール一覧から選択（self）

```
/role self
```

ロールを指定せずに実行すると、サーバーのロール一覧が表示され、一括でボタンを作成できます。

## ユースケース

### ケース1: セッション終了時のロール変更

1. `/role change` でPLロール→通過者ロールのボタンを作成
2. セッション終了時にボタンを押す
3. 全PLのロールが一括で変更される

### ケース2: 参加希望の受付

1. `/role self 付与するロール:@参加希望` を実行
2. 参加希望者がボタンを押してロールを取得
3. GMが参加者を把握できる

## 注意事項

::: warning 注意
「ボタンを他の人からも見えるようにする」で「はい」を選択した場合：
- Discordを閉じてもボタンが残り続けます
- `/copy` で複製が可能になります
- 誰でもボタンを押せるため、設置チャンネルに注意してください
:::

::: tip ヒント
マダミナリンクより上位のロールは操作できません。
:::
