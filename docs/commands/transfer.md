# /transfer

PDFなどのファイルやメッセージを別のチャンネルに転送するボタンを作成します。

## 使用方法

```
/transfer
```

## オプション

| オプション | 必須 | 説明 |
|-----------|------|------|
| 転送先 | いいえ | 転送先のチャンネル（指定しない場合はリストから選択） |

## 使用例

### 転送先を指定

```
/transfer 転送先:#キャラAの部屋
```

転送用ボタンが作成されます。

### リストから選択

```
/transfer
```

転送先を指定せずに実行すると、同じカテゴリ内のチャンネルリストが表示され、選択できます。

## 転送の手順

1. 転送したいファイル・メッセージを用意
2. `/transfer` で転送ボタンを作成
3. 転送したいメッセージに任意のリアクション（絵文字）をつける
4. 転送ボタンに同じリアクションをつける
5. ボタンを押すと転送が実行される

## ユースケース

### ケース1: 個別情報の配布

1. 共通情報チャンネルに各プレイヤーの個別情報を用意
2. 各個別チャンネルへの転送ボタンを作成
3. 該当する情報にリアクションをつけてボタンで転送
4. ピン留めされた状態は転送先でも維持される

### ケース2: PDFファイルの配布

1. GMチャンネルにPDFファイルを用意
2. プレイヤーチャンネルへの転送ボタンを作成
3. リアクションで選択して転送

## 注意事項

::: warning 注意
転送先をリストから選択する場合、同じカテゴリ内のチャンネルのみ表示されます。別カテゴリに転送したい場合は、転送先オプションで直接指定してください。
:::

::: tip ヒント
- 転送元でピン留めされたメッセージは、転送先でもピン留めされます
- 他の転送ボタンに同じリアクションがついていても、そちらには影響しません
:::
