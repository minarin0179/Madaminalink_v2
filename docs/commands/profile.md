# /profile

Botのサーバープロフィール（ニックネームやアバター）を変更します。

## 使用方法

```
/profile set
```

## サブコマンド

| サブコマンド | 説明 |
|-------------|------|
| set | Botのサーバープロフィールを設定 |
| reset | プロフィールをすべてリセット |

## 使用例

### プロフィールを設定

```
/profile set
```

モーダルが表示され、ニックネームを設定できます。

### プロフィールをリセット

```
/profile reset
```

設定したニックネーム、アバター、バナーをすべてリセットします。

## ユースケース

### ケース1: サーバーごとにBotの名前を変更

1. `/profile set` を実行
2. モーダルでニックネームを入力
3. サーバー内でのBot表示名が変更される

## 注意事項

::: warning 注意
このコマンドは管理者権限が必要です。
:::
