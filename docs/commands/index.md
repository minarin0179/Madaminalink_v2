# コマンド一覧

マダミナリンクで使用できるコマンドの一覧です。

## セットアップ系

| コマンド | 説明 |
|---------|------|
| [/setup](/commands/setup) | 新規プレイ用のカテゴリ、チャンネル、ロールを一括作成 |
| [/role](/commands/role) | ロールの付与・解除を行うボタンを作成 |

## 進行管理

| コマンド | 説明 |
|---------|------|
| [/open](/commands/open) | チャンネルを特定のロールに対してボタン操作で公開 |
| [/gather](/commands/gather) | 他のVCにいるユーザーを、自分のいるVCに一括で移動 |
| [/dice](/commands/dice) | ダイスを振るボタンを設置 |
| [/poll](/commands/poll) | キャラ選択や犯人投票を行うための投票機能を作成 |
| [/remind](/commands/remind) | 指定した日時にメッセージを自動送信するリマインダーを設定 |
| [/order](/commands/order) | ランダムでユーザーの順番決めを行う |

## ログ・アーカイブ

| コマンド | 説明 |
|---------|------|
| [/archive](/commands/archive) | テキストチャンネルのログをスレッド形式で保存 |
| [/transfer](/commands/transfer) | PDFなどのファイルやメッセージを別のチャンネルに転送 |
| [/log](/commands/log) | プレイ後のチャンネルを非表示にしてログとして残す |

## クリーンアップ

| コマンド | 説明 |
|---------|------|
| [/cleanup](/commands/cleanup) | チャンネルまたはカテゴリ内のメッセージをすべて削除 |
| [/delete](/commands/delete) | カテゴリとその中のチャンネルをすべて削除 |
| [/rename](/commands/rename) | ニックネームを一括で変更またはリセット |

## その他

| コマンド | 説明 |
|---------|------|
| [/copy](/commands/copy) | チャンネルやカテゴリを複製 |
| [/sync](/commands/sync) | カテゴリ内のすべてのチャンネルの権限をカテゴリと同期 |
| [/server](/commands/server) | サーバーのメンバー数、ロール数、チャンネル数を確認 |
| [/profile](/commands/profile) | Botのサーバープロフィール（ニックネームやアバター）を変更 |
| [/ping](/commands/ping) | マダミナリンクの稼働状況を確認 |
