# コマンド一覧

マダミナリンクで使用できるコマンドの一覧です。

## セットアップ系

| コマンド | 説明 |
|---------|------|
| [/setup](/commands/setup) | プレイ用のカテゴリー、チャンネル、ロールを一括作成 |
| [/role](/commands/role) | ロールの付け外しができるボタンを表示 |

## 進行管理

| コマンド | 説明 |
|---------|------|
| [/open](/commands/open) | ボタンを押すことでチャンネルを公開 |
| [/gather](/commands/gather) | 他のVCにいるユーザーを自分のいるVCに移動 |
| [/dice](/commands/dice) | ダイスを振るボタンを表示 |
| [/poll](/commands/poll) | 投票を作成・集計 |
| [/remind](/commands/remind) | リマインダーを設定 |
| [/order](/commands/order) | 順番を管理 |

## ログ・アーカイブ

| コマンド | 説明 |
|---------|------|
| [/archive](/commands/archive) | チャンネルをスレッドにして保存 |
| [/transfer](/commands/transfer) | メッセージを別のチャンネルに転送 |
| [/log](/commands/log) | ログを取得 |

## クリーンアップ

| コマンド | 説明 |
|---------|------|
| [/cleanup](/commands/cleanup) | チャンネルのメッセージをすべて削除 |
| [/delete](/commands/delete) | カテゴリー内のチャンネルをすべて削除 |
| [/rename](/commands/rename) | ニックネームをまとめてリセット |

## その他

| コマンド | 説明 |
|---------|------|
| [/copy](/commands/copy) | チャンネルやカテゴリーを複製 |
| [/sync](/commands/sync) | チャンネルの権限を同期 |
| [/server](/commands/server) | サーバー情報を表示 |
| [/profile](/commands/profile) | プロフィールを設定 |
| [/ping](/commands/ping) | Botの応答を確認 |
