## 使用方法

```
/copy
```

## オプション

| オプション | 必須 | 説明 |
|-----------|------|------|
| 対象 | いいえ | コピーするチャンネルまたはカテゴリ（デフォルト: コマンドを入力したチャンネル） |
| メッセージを複製しない | いいえ | 「はい」を選択するとチャンネルのみ複製され、メッセージは複製されない |

## 使用例

### 現在のチャンネルを複製

```
/copy
```

コマンドを入力したチャンネルを複製します。

### 特定のカテゴリを複製

```
/copy 対象:#セッション用
```

カテゴリとその中のチャンネルをすべて複製します。

<!-- TODO: 複製前後のスクリーンショット -->

### メッセージなしで複製

```
/copy 対象:#セッション用 メッセージを複製しない:はい
```

チャンネル構造のみを複製し、メッセージはコピーしません。

## 用途

### ケース1: テンプレートカテゴリの活用

1. テンプレートとなるカテゴリを作成
2. 新しいセッションの際に `/copy` で複製
3. 複製されたカテゴリを使用してセッションを開始

## 注意事項

::: warning 注意
通知が発生するため、サーバーの通知設定を「@mentionsのみ」にすることを推奨します。
:::

::: tip ヒント
`/transfer` や `/open` で作成したボタンの対象を新しいチャンネルに変更したい場合は、同じカテゴリに入れてからコピーしてください。
:::
