## 使用方法

```
/server
```

## オプション

オプションはありません。

## 使用例

### サーバー情報を確認

```
/server
```

以下のような情報が表示されます：

```
メンバー数 : XX人 (+ BotYY台)
ロール数 : ZZ個 (上限250個)
チャンネル数 : WW個 (上限500個)
```

## 用途

### ケース1: リソース上限の確認

1. `/server` を実行
2. チャンネル数やロール数が上限に近づいていないか確認
3. 必要に応じて `/delete` などで整理

## 注意事項

::: tip ヒント
コマンドを実行した人にだけ情報が表示されます（Ephemeral）。
:::
