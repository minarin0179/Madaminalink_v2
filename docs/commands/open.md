## 使用方法

```
/open 公開相手:@ロール名
```

## オプション

| オプション | 必須 | 説明 |
|-----------|------|------|
| 公開相手 | はい | チャンネルを公開するロールまたはメンバー |
| チャンネル | いいえ | 公開するチャンネル（デフォルト: コマンドを入力したチャンネル） |

## 使用例

### 基本的な使い方

```
/open 公開相手:@プレイヤー
```

ボタンを押すと、指定したロールにチャンネルが公開されます。

### 特定のチャンネルを公開

```
/open 公開相手:@プレイヤー チャンネル:#秘密の部屋
```

指定したチャンネルを公開するボタンを作成します。

## 用途

### ケース1: 追加情報の段階的公開

1. 公開したいチャンネルを事前に非表示に設定
2. `/open` でボタンを作成
3. 適切なタイミングでボタンを押してチャンネルを公開

### ケース2: 個別情報の公開

1. 各プレイヤーの個別チャンネルを用意
2. プレイヤーロールに対して `/open` ボタンを作成
3. ゲーム開始時にボタンを押して一斉公開

## 注意事項

::: tip ヒント
チャンネルを指定せずに実行すると、コマンドを実行したチャンネルが対象になります。
:::
