# /log

プレイ後のチャンネルを非表示にしてログとして残します。カテゴリ名に日付を追加し、閲覧用ロールのみが見られるようにします。

## 使用方法

```
/log 非表示にするカテゴリ:#カテゴリ名
```

## オプション

| オプション | 必須 | 説明 |
|-----------|------|------|
| 非表示にするカテゴリ | はい | 非表示にするカテゴリ |
| 閲覧用ロール | いいえ | このロールを持つメンバーのみ閲覧可能にする |
| ボイスチャンネルを削除する | いいえ | カテゴリ内のVCを削除するか（デフォルト: はい） |
| 日付をカテゴリ名に追加する | いいえ | 最後のメッセージ日付をカテゴリ名に追加（デフォルト: はい） |

## 使用例

### 基本的な使い方

```
/log 非表示にするカテゴリ:#終了したセッション
```

カテゴリを非表示にし、カテゴリ名に日付を追加します。

### 閲覧用ロールを指定

```
/log 非表示にするカテゴリ:#終了したセッション 閲覧用ロール:@ログ閲覧者
```

指定したロールを持つメンバーのみがログを閲覧できるようになります。

### ボイスチャンネルを残す

```
/log 非表示にするカテゴリ:#終了したセッション ボイスチャンネルを削除する:いいえ
```

VCを削除せずにログとして残します。

## 用途

### ケース1: セッション終了後のログ化

1. セッションが終了したカテゴリを選択
2. `/log` を実行
3. カテゴリが非表示になり、日付が追加される
4. 必要に応じて閲覧用ロールで限定公開

## 注意事項

::: tip ヒント
各チャンネル名には、最も多くメッセージを送信したメンバーの名前が追加されます（条件を満たした場合）。
:::
