## 使用方法

```
/ping
```

実行すると、マダミナリンクの稼働状況と応答時間が表示されます。

![「マダミナリンクは現在稼働中です (173ms)」という応答メッセージ](../images/commands/ping/ping-response.png)

::: tip DMでも使用可能
このコマンドはサーバー内だけでなく、マダミナリンクへのDM（ダイレクトメッセージ）でも使用できます。
:::

::: warning 応答がない場合
コマンドを実行しても応答がない場合、マダミナリンクに障害が発生している可能性があります。
:::

## 用途

### Botの動作確認

マダミナリンクが正常に動作しているか確認したいときに使用します。

- 他のコマンドが反応しないときの切り分けに有効
