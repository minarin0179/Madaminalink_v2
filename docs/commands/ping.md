## 使用方法

```
/ping
```

## オプション

オプションはありません。

## 使用例

### 稼働確認

```
/ping
```

マダミナリンクが稼働中であれば、応答時間とともにメッセージが表示されます。

```
マダミナリンクは現在稼働中です (XXms)
```

## 用途

### ケース1: Botの動作確認

1. マダミナリンクが応答しない場合に `/ping` を実行
2. 応答があれば正常に稼働中
3. 応答がなければBotに問題が発生している可能性あり
