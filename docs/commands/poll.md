# /poll

キャラ選択や犯人投票を行うための投票機能を作成します。

## 使用方法

```
/poll 投票モード:キャラ選択 選択肢:選択肢1 選択肢2 選択肢3
```

## オプション

| オプション | 必須 | 説明 |
|-----------|------|------|
| 投票モード | はい | 「キャラ選択」または「犯人投票」 |
| 選択肢 | はい | 選択肢をスペース区切りで入力 |

## 投票モードの違い

- **キャラ選択**: プレイヤーがキャラクターを選択する際に使用
- **犯人投票**: 集計結果をPLに非表示にし、得票数を表示

## 使用例

### キャラ選択

```
/poll 投票モード:キャラ選択 選択肢:探偵 助手 医者 執事
```

プレイヤーがキャラクターを選択できるボタンが表示されます。

### 犯人投票

```
/poll 投票モード:犯人投票 選択肢:Aさん Bさん Cさん
```

犯人を投票するボタンが表示されます。集計結果はGMのみ確認可能です。

### ロールを選択肢に使用

```
/poll 投票モード:キャラ選択 選択肢:@探偵 @助手 @医者
```

ロールをメンションすると、ロール名が選択肢として使用されます。

## 注意事項

::: warning 注意
選択肢は最大25個まで設定できます。また、各選択肢の文字数には制限があります。
:::

::: tip ヒント
投票は15分後に自動的に締め切られます。
:::
