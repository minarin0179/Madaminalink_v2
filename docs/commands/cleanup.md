## 使用方法

```
/cleanup 対象:#一般
```

実行すると、指定したチャンネル内のすべてのメッセージが削除されます。

対象を指定しない場合、コマンドを入力したチャンネルのメッセージが削除されます。

```
/cleanup
```

::: danger 削除したメッセージは復元できません
実行前に必要なログがないか確認してください。ログを残したい場合は [/archive](/commands/archive) で保存できます。
:::

::: warning 荒らし対策のため、Botがサーバーに追加されてから24時間は使用できません
:::

::: tip 削除の仕組み
14日以内のメッセージは100件ずつ削除できるため高速ですが、14日以上前のメッセージは1件ずつ削除されるため、古いメッセージが多いと時間がかかります。
:::

## オプション

| オプション | 必須 | 説明 |
|-----------|------|------|
| 対象 | いいえ | 削除対象のチャンネルまたはカテゴリ |

### オプションの詳細

#### 対象

削除対象のチャンネルまたはカテゴリを指定します。

- **デフォルト**: コマンドを入力したチャンネル
- **指定可能な対象**: テキストチャンネル、カテゴリ

カテゴリを指定した場合、カテゴリ内のすべてのテキストチャンネル・ボイスチャンネルのメッセージが削除されます。


## 使用例

### 現在のチャンネルのメッセージを削除

```
/cleanup
```

コマンドを入力したチャンネルのメッセージをすべて削除します。

### 特定のチャンネルを指定

```
/cleanup 対象:#一般
```

指定したチャンネルのメッセージをすべて削除します。

### カテゴリ全体を対象にする

```
/cleanup 対象:セッション用
```

カテゴリ内のすべてのテキストチャンネル・ボイスチャンネルのメッセージが一括削除されます。

## 用途

### セッション終了後にカテゴリを再利用する

セッションが終了し、次回のセッションに向けてカテゴリ内のチャンネルをリセットする場合に使用します。

1. 必要であれば `/archive` でログを保存
2. `/cleanup 対象:セッション用` を実行
3. 全チャンネルのメッセージが削除され、再利用可能な状態に

::: tip 範囲を指定して削除したい場合
すべてではなく一部のメッセージだけを削除したい場合は、メッセージを右クリック→「アプリ」→「[これ以前を削除](/commands/delete-before)」/「[これ以降を削除](/commands/delete-after)」を使用できます。
:::
