# /cleanup

チャンネルまたはカテゴリ内のメッセージをすべて削除します。シナリオ開始前の状態に戻してカテゴリを再利用する際に便利です。

## 使用方法

```
/cleanup
```

## オプション

| オプション | 必須 | 説明 |
|-----------|------|------|
| 対象 | いいえ | 削除対象のチャンネルまたはカテゴリ（デフォルト: コマンドを入力したチャンネル） |

## 使用例

### 現在のチャンネルのメッセージを削除

```
/cleanup
```

コマンドを入力したチャンネルのメッセージをすべて削除します。

### 特定のチャンネルを指定

```
/cleanup 対象:#一般
```

指定したチャンネルのメッセージをすべて削除します。

### カテゴリ全体を対象にする

```
/cleanup 対象:#セッション用
```

カテゴリを指定すると、カテゴリ内のすべてのテキストチャンネルのメッセージが削除されます。

## ユースケース

### ケース1: カテゴリを再利用する

1. 前回のセッションで使用したカテゴリを選択
2. `/cleanup 対象:#カテゴリ名` を実行
3. 全チャンネルのメッセージが削除され、再利用可能な状態に

## 注意事項

::: danger 危険
削除したメッセージは復元できません。実行前に必要なログがないか確認してください。
:::

::: tip ヒント
範囲を指定して削除したい場合は、メッセージを右クリック→「アプリ」→「これ以前を削除」/「これ以降を削除」を使用できます。
:::
