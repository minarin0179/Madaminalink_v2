# /gather

他のVCにいるユーザーを、自分のいるVCに一括で移動させます。密談で分かれた人を全体会議に呼び戻す際などに便利です。

## 使用方法

```
/gather
```

## オプション

| オプション | 必須 | 説明 |
|-----------|------|------|
| 対象 | いいえ | 指定したロールを持つメンバーのみを移動対象にする（デフォルト: @everyone） |

## 使用例

### 基本的な使い方

```
/gather
```

GMがいるVCに、サーバー内の他のVCにいる全メンバーを集めます。

### 特定のロールのメンバーだけを集める

```
/gather 対象:@プレイヤー
```

指定したロールを持つメンバーのみを移動させます。

## ユースケース

### ケース1: 密談から全体会議に戻す

1. GMが全体会議用のVCに移動する
2. `/gather` を実行する
3. 他のVCにいたプレイヤーが全員、GMのいるVCに移動される

## 注意事項

::: warning 注意
GMがVCに参加していない状態では使用できません。必ずVCに入ってから実行してください。
:::

::: tip ヒント
閲覧権限がないチャンネルにいるメンバーも強制的に移動させることができます。
:::
