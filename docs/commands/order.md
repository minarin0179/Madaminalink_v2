## 使用方法

```
/order
```

## オプション

| オプション | 必須 | 説明 |
|-----------|------|------|
| ユーザーまたはロール | いいえ | 初期メンバーとして追加するユーザーやロール |
| ボイスチャンネル | いいえ | VCにいるメンバーを自動追加 |

## 使用例

### 参加ボタンで集める

```
/order
```

参加ボタンが表示され、押したメンバーが追加されます。シャッフルボタンで順番を決定します。

<!-- TODO: 順番決めの画面スクリーンショット -->

### ロールで指定

```
/order ユーザーまたはロール:@プレイヤー
```

指定したロールを持つメンバーが初期メンバーとして追加されます。

### VCのメンバーを追加

```
/order ボイスチャンネル:#全体会議
```

指定したVCにいるメンバーを自動的に追加します。

## 用途

### ケース1: セッション開始時の行動順決め

1. `/order` を実行
2. プレイヤーが参加ボタンを押す
3. 全員揃ったらシャッフルボタンで順番を決定

## 注意事項

::: tip ヒント
参加ボタンを押すことで、後からメンバーを追加できます。
:::
